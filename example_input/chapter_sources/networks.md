# Computer Networks

## Topologies

In a star topology, each computer connects separately to a central device such as a switch or a hub. Because each device has its own connection, one cable failure usually affects only one computer. This makes troubleshooting easier compared with a bus topology, where all devices share a single backbone cable.

Two advantages of a star topology are: (1) it is easy to identify a failed node, and (2) a failure in one cable usually does not affect the rest of the network.

## LAN, WAN, MAN

A LAN (Local Area Network) covers a small area such as a classroom or office. A WAN (Wide Area Network) covers a large geographic area and often uses leased lines or the public internet.

## Network devices

A switch forwards packets only to the port of the destination device using MAC addresses. A router forwards packets between different networks using IP addresses.

## Protocols

The TCP/IP protocol stack is the foundation of the public internet. TCP provides reliable, ordered delivery. IP provides addressing and routing between networks.
