# Databases

## Primary keys

A primary key is a column (or set of columns) whose values uniquely identify each row in a table. A primary key cannot be null and must be unique across all rows.

## Foreign keys

A foreign key is a column in one table that references the primary key of another table. Foreign keys enforce referential integrity between related tables.

## Normalization

Normalization is the process of organizing data to reduce redundancy. First normal form (1NF) requires atomic values in every cell. Second normal form (2NF) requires that all non-key attributes depend on the whole primary key, not just part of it.

## SQL basics

SELECT retrieves rows from one or more tables. INSERT adds new rows. UPDATE modifies existing rows. DELETE removes rows. WHERE filters which rows are affected.
