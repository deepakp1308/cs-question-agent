# Networks — Mark Scheme

## Q1 Star topology advantages (2 marks)

Accept any two of:
- Easy to identify a failed node / fault isolation is simple.
- A single cable failure does not bring down the rest of the network.
- Easy to add or remove devices without disrupting others.
- Centralized management of devices through the switch.

## Q2 Switch vs Router (2 marks)

Must include:
- Switch forwards within a single network using MAC addresses.
- Router forwards between different networks using IP addresses.
